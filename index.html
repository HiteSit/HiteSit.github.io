Extract dock scores from Autodock GPU output-files while making a conformational search of the ligands

Fast visualize the output scores of a docking protocol is the key feature to best analyze the results, this script will take all the informations about the dock-runs and summarize them into a tidy, well understandable, table while making a conformational search both of the ligand' global minimum and the actual scored pose.
